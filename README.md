# House Price Prediction

## Overview

This project predicts house prices using Machine Learning techniques.

The project includes:

- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Linear Regression
- Random Forest Regression
- Model Evaluation
- Data Visualization

## Dataset

Housing Prices Dataset from Kaggle.

## Libraries Used

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Models

- Linear Regression
- Random Forest Regressor

## Evaluation Metrics

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score
